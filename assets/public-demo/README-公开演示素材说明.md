# 公开演示素材说明

本目录用于放公开安全的演示素材。

第一阶段暂不批量下载素材，只建立规则。

允许放入：

- CC0 剪影角色
- 自制几何角色
- CSS 或 SVG 生成的抽象图形
- 授权明确的图标和纹理

当前已登记的公开安全自制角色：

- `public-safe-css-boxbot`：CSS 生成的纸箱机器人，只保留“纸盒 / 几何 / 黑圆眼”的通用视觉方向，不使用任何现有商业角色名称、图片或精确造型。

禁止放入：

- 真实游戏 / 动漫 / 影视 IP 角色图
- 真实账号私有素材
- 授权不明的网络图片
- 无法说明来源的截图

素材正式入库前，必须先写入：

- `数据/公开素材候选表.csv`
- `数据/已入库素材清单.json`
