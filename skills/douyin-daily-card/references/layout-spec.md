# Layout Spec

This spec fixes the current public-safe white-tech daily-report template.

## Canvas

- Ratio: `3:4`.
- Page size: `1080 x 1440 px`.
- Page structure: one cover page plus as many content pages as needed.
- Standard filename pattern: `01-cover.png`, `02-page-02.png`, `03-page-03.png`.

## Brand Defaults

- Account name: `社区服务中心`.
- Account ID: `community-demo-001`.
- Cover title: `社区公告日报来了`.
- Template: `public-safe-v1-white-tech`.
- Output name pattern: `generic-daily-report-YYYY-MM-DD`.

## Typography

- Chinese stack: `PingFang SC`, `Hiragino Sans GB`, `Microsoft YaHei`, `Noto Sans CJK SC`, `sans-serif`.
- Number stack: `Menlo`, `Consolas`, `monospace`.
- Body: `31px`, line-height `1.54`, font-weight `650`.
- Section title: `34px`, font-weight `900`.
- Subhead: `34px`, font-weight `900`.

## Content Area

- Left: `126px`.
- Top: `228px`.
- Width: `828px`.
- Height: `948px`.
- Max pagination height: `948px`.
- Page number: left `126px`, bottom `76px`.

## Cover Positions

- Account: `x=81`, `y=108`, size `40`.
- Main title: `x=113`, `y=368`, size `78`.
- Date: `x=328`, `y=708`, size `58`.
- Public ID: `x=691`, `y=1312`, size `24`.

## Cover Rules

The cover is a title/date card. Do not place the article summary, keywords, or body excerpt on the cover by default.
