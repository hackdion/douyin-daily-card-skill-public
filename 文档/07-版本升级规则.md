# 07-版本升级规则

本项目会持续升级，所以必须区分稳定版和实验版。

## 版本命名

模板版本：

```text
weekday-cover-base-v1
weekday-cover-base-v2-beta
comic-cover-v1
magazine-cover-v1
```

模块版本：

```text
comic-halftone-background-v1
radar-hud-frame-v1
magazine-title-lockup-v1
```

## 状态规则

`draft`：

- 只有想法。
- 不能给用户正式使用。

`beta`：

- 可以手动指定测试。
- 不能自动默认使用。

`stable`：

- 可以被 agent 自动选择。
- 必须有示例输出和验证结果。

`deprecated`：

- 旧模板。
- 保留兼容，但不推荐使用。

## 升级原则

- 新版本不能覆盖旧版本。
- 旧版本能继续渲染。
- 每次升级要说明解决了什么问题。
- 新模板进入 `stable` 前必须经过至少一次真实任务验证。

## 公开发布前检查

- 没有高风险素材。
- 没有私有账号信息。
- 没有真实第三方 IP 角色图。
- 所有需要署名的素材都已署名。
- README 和文档仍然中文优先。
